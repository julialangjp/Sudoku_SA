# Sudoku_SA
数独をSimulatedAnnealingで解く  
  
## Usage
### コンソールで実行
```julia
> julia Sudoku_SA.jl
```

### 結果出力例
  
[5, 3, 4, 6, 7, 8, 9, 1, 2]  
[6, 7, 2, 1, 9, 5, 3, 4, 8]  
[1, 9, 8, 3, 4, 2, 5, 6, 7]  
[8, 5, 9, 7, 6, 1, 4, 2, 3]  
[4, 2, 6, 8, 5, 3, 7, 9, 1]  
[7, 1, 3, 9, 2, 4, 8, 5, 6]  
[9, 6, 1, 5, 3, 7, 2, 8, 4]  
[2, 8, 7, 4, 1, 9, 6, 3, 5]  
[3, 4, 5, 2, 8, 6, 1, 7, 9]  
energy = 0, temp = 0.7272854775176761  
OK  
22.3284739 sec  
  
「energy = 0」の場合が、正解が得られています。  
SA法では近似解を求めているので、必ずしも正解でない場合も出てきます。  
何度か繰り返すことで、「energy = 0」となるパターンを得ることができます。  

## 備考
  
　パラメータは、サンプルデータを解くために調整したものです。他の問題の場合には、パラメータを調整する必要があります。
